# FernandesLAB

Website for the **Fernandes Lab** — quantitative genetics, statistics & AI for plant
breeding at the University of Arkansas.

🌐 **Live:** https://fernandeslab.org

A static site (plain HTML + one CSS file, no build step). Preview it locally with:

```bash
python3 -m http.server 8000   # then open http://localhost:8000
```

Maintenance notes — structure, editing, theming, the Google Scholar publications
updater, and deployment — are in [`CLAUDE.md`](CLAUDE.md).
